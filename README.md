# traildb.tr

TrailDB bindings for [Crystal](http://crystal-lang.org/).

Check [traildb](https://github.com/traildb/traildb) for general db driver documentation.


## Installation

Add this to your application's `shard.yml`:

```yaml
dependencies:
  traildb:
    github: maiha/traildb.cr
```


## Usage


```crystal
require "traildb"

```
